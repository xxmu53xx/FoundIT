# Garbage Management System

## Product Description
The Garbage Management System is a comprehensive solution designed to optimize waste collection, enhance communication between residents and waste collectors, and promote sustainable waste management practices. Our platform offers real-time updates, smart notifications, and intuitive tools to efficiently manage garbage disposal schedules and locations.

## Functional Requirements

| ID | Requirement | Description |
|----|------------|-------------|
| F1 | Online Schedule Viewing | Users can access interactive calendars showing exact times and dates for garbage collection in their area |
| F2 | Interactive Garbage Map | System displays a map showing locations of reported garbage piles with status indicators |
| F3 | Push Notifications | Mobile app sends timely alerts about upcoming pickups, delays, and nearby collection trucks |
| F4 | Feedback System | Users can submit feedback or complaints regarding service issues |
| F5 | Waste Reduction Tips | Educational content to help users improve waste management practices |
| F6 | Pickup Reminders | Location-based reminders for scheduled trash collection |
| F7 | User Registration | Secure account creation and login for accessing personalized features |
| F8 | Collection History | Maintenance of past garbage collection records for user reference |
| F9 | Analytics Dashboard | Display of key waste management statistics for users and service providers |
| F10 | Missed Collection Reporting | Feature allowing users to report if their garbage wasn't collected as scheduled |

## Non-Functional Requirements

| ID | Requirement | Description |
|----|------------|-------------|
| NF1 | Scalability | Support for up to 100 concurrent users without performance degradation |
| NF2 | Data Security | End-to-end encryption for all stored and transmitted data |
| NF3 | Privacy | Secure storage of user data with no third-party sharing |
| NF4 | Availability | System uptime of at least 99.9% |
| NF5 | Data Backup | Automated database backups performed every 24 hours |

## Getting Started

### Prerequisites
- Node.js (v16.0.0+)
- npm (v8.0.0+) or yarn (v1.22.0+)
- Android Studio (for mobile development)
- Firebase account with Blaze plan (for full functionality)

### Installation

#### Web Application Setup
```bash
# Clone the repository
git clone https://github.com/yourusername/IT342-G4-GarbageManagementSystem.git

# Navigate to project directory
cd IT342-G4-GarbageManagementSystem

# Install dependencies
npm install
# or
yarn install

# Start development server
npm run dev
# or
yarn dev
```

## Mobile Application Setup

1. Open Android Studio
2. Select "Open an existing project"
3. Navigate to the mobile directory in the cloned repository
4. Install any required Android SDK components when prompted
5. Connect a device or start an emulator
6, Click the "Run" button to build and deploy the application

## Firebase Configuration

1. Create a new Firebase project at Firebase Console
2. Enable Authentication, Firestore Database, and Storage services
3. Add a Web app and Android app to your Firebase project
4. Copy the configuration details to your respective applications
5. Set up security rules for Firestore and Storage
